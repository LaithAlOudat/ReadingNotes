## Chapter 3 (Lists):
#### Lists are elements in html we use to create a certain list that contains a particular numbers of list items.
#### There are two types of lists in html language:
##### 1) <ol></ol>: Ordered list: this list will listed a specific number of list item and they will begin with numbers. 1. 2. 3. And so on.
##### 2) <ul></ul>: Unordered list: this list will listed a specific number of list item and they will start with dots or bullets.
#### <li></li>: list item: this element can’t come alone. It must contained in an <ol> or <ul> and doesn’t matter if you close it with closing tag or not because the browser know when it’s end by starts another <li> element.
#### Nested lists: In a particular situations you will need a list inside another list so this is called nested lists.
