##### There are many types of forms in html:
###### 1) Adding text: such as text input, password input and text area.
###### 2) making choices: such as radio buttons, checkboxes and drop-down boxes.
###### 3) Submitting forms: like submit buttons, image buttons and file upload.
##### How forms work?
###### 1) a user fills in a form and then presses a button to submit the information to the server.
###### 2) the name of each from control is sent to the server along with the value the user or selects.
###### 3) The server processes the information using a programming language such as PHP, C#, VB.net, or java. It may also store the information in a database.
###### 4) the server creates a new page to send back to the browser based on the information received.
