### Chapter 16 (Images):
##### Controling the size and alignment of the images using CSS keeps rules that affect the presentation of your page in the CSS and out of the HTML markup.
##### To control the size of images in CSS you can determine the width and height in the CSS file. And for the alignment you can use float, margin and position properities.
##### you can set an image as a background rather than colors with background-image:url propriety in the CSS file for the element that you want to set the background for.
##### repeating image: The background-repeat property has four values: 
###### 1) repeat: the background image is repeated both horizontally and vertically.
###### 2) repeat-x: The image is repeated horizontally only.
###### 3) repeat-y: The image is repeated vertically only.
###### 4) no-repeat: The image is only shown once. 
##### types of background:
###### 1) background-color.
###### 2) background-image.
###### 3) background-repeat.
###### 4) background-attachment.
###### 5) background-position.
